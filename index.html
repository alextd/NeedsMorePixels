Needs More Pixels

(The game from Um, Actually)

Click image to step to the next clearer image. Use the slider bar beneath image to select image clarity. Arrow keys also work: up to start, down to end, left/right step one by one. Select a local image, or paste a URL to play a new game.

For optimal mobile party experience: 1. Share the URL with everyone 2. Save this page to your home screen (to run without browser UI getting in the way) 3. Keep phone horizontal 4. Select image 5. Display to your TV for everyone to see Known issues: Sometimes the image scales badly (when run from the home screen, after rotating the phone). Rather than fix that, there is a Restart button to re-align the image.
